反转链表

Posted on 2020-03-31 | In Leetcode

题目描述

  • 反转一个单链表。

    示例:

知识点

递归

  • 循环调用函数自身，缩小计算范围

  • 必须要有递归结束条件，否则会无限递归

  • 递归是从内到外进行数据计算

    • 阶乘

    • int foo(int n){
  if (n == 1)
    return 1;
  else 
    return n * foo(n-1)
} 
<!--￼1-->

  • 结果如下，输出为[1, 2, 3]

  • 理解

    1. 函数进来之后先遍历链表，当head为2时 head.next 为空返回

    2. 此时cur 为链表的头节点（第一个节点）

    3. 将3->2->None 进行翻转

    4. 之前每次的递归调用都需要返回，有始有终，进来了自然需要出去。

    5. reverseList(head->1)
    reverseList(head->2)
        reverseList(head->3)
            reverseList(head->4)
                reverseList(head->5)
                    触发终止条件，返回head
                cur = reverseList()返回结果
                cur->5
                head->next->next = head
                // 5.next->4
                返回cur(5)

            cur = reverseList()返回结果
            cur->5
            head->next->next = head
            // 4.next->3
            返回cur(5)

        cur = reverseList()返回结果
        cur->5
        head->next->next = head
        // 3.next->2
        返回cur(5)

    cur = reverseList()返回结果
    cur->5
    head->next->next = head
    // 2.next->1
    返回cur(5)
